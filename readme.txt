URL SHORTNER:

This service takes a valid url and returns a short version of that url.
Also keeps a track of total visit/clicks on the url

Routes:

1> POST/URL - Generate a new short URL and returns the shortned URL in the format exp.com/random-id

2> GET/:ID -  Redirects to the original URL

3> GET/URL/ANALYTICS/:ID - Returns the clicks for the provided short id
